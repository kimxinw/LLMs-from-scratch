# LLMS from scratch 从零构建大模型

从零开始实现GPT架构的简易大模型.

## 特性

- Dataset processing 数据处理
- Multi-head self-attention 多头自注意力机制
- GPT architecture GPT架构
- Training pipeline 模型训练

## 项目结构

```text
LLMs-from-scratch/
├── dataset.py      数据处理
├── attention.py    多头注意力机制
├── gpt.py          GPT架构
├── train.py        模型训练
├── the-verdict.txt 样本数据集
└── README.md
```
